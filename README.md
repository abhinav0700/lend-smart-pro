# Mazhilini Finance - Lending Management System

This repository contains the front-end for Mazhilini Finance, a lending management application for tracking loans, customers and payments.

Quick start
1. Install dependencies

```bash
npm install
```

2. Start development server

```bash
npm run dev
```

Project structure
- `src/` - React + TypeScript source files
- `public/` - static assets (favicon, images)

Notes
- If you previously had third-party tooling references (e.g. Lovable) those have been removed and the repository is ready for local development.
- If you see references to external placeholder images, replace them with your own assets in `public/` and update `index.html` accordingly.
